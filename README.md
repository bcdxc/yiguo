## 项目简介

- A wechat program of "easy pot food" and the business management system based on Vue.

- 该项目运行在本地。

- vue-yiguo为web端项目，wx-yiguo为小程序端项目。

- 小程序使用原生框架和 iView 组件库搭建，web端使用 Vue 和 ElementUi 搭建。运行 json-server 服务，端口号为 3006。
